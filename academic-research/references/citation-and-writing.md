# Citation and Writing

## Citation Integrity

Every citation used in drafting should pass this minimum standard:

- the source is real
- the source has been directly examined, not inferred from another summary
- the claim being cited matches what the source actually supports
- enough metadata is known to identify the source correctly

If any of these fail, do not cite the source as if it were verified.

## Citation Ledger

Maintain a traceable ledger during writing.

```text
Citation key:
Full reference or verified metadata:
What this source supports:
Where it will be cited:
Confidence in match:
Notes:
```

## Summary Standard For Retained References

For every reference that survives into the evidence pack, record:

- problem studied
- method or design
- key finding
- limitation
- relevance to the current paper

This is mandatory when the user asks for real references or literature support.

## Writing From Evidence

Before drafting a section, map each paragraph to:

- point
- supporting references
- analytical role, such as background, contrast, method justification, or limitation

Do not write paragraphs first and hunt for citations later.

When drafting in Chinese, make sure citation placement follows the logic of the claim. Do not place all citations mechanically at the end of long paragraphs if different sentences rely on different sources.

## Section Discipline

Infer the exact structure from the field, but common expectations include:

- introduction: motivate the problem, establish the gap, state the contribution
- related work or literature review: compare prior work, show the missing space, position the paper
- methods: explain the design precisely enough to evaluate credibility
- data or experimental setup: show what evidence exists and how it is used
- results: report findings without overstating them
- discussion: interpret the findings and connect them back to the field
- limitations: state real weaknesses
- conclusion: summarize the takeaways without introducing new unsupported claims

## Prohibited Shortcuts

Do not:

- cite papers you have not actually checked
- use fake page numbers or DOI values
- attribute a claim to a source that only vaguely discusses it
- use one paper as proof for a stronger claim than it makes
- convert weak evidence into strong causal language
- let polished Chinese wording hide weak evidence linkage
