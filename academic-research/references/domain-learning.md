# Domain Learning

## Purpose

Use this reference at the start of any discipline-specific request. The goal is to learn the field efficiently enough to make good downstream decisions without drowning in irrelevant material.

## Source Priority

Prefer sources in roughly this order:

1. survey papers, review articles, handbooks, and canonical textbooks
2. top-tier journals or conferences in the target field
3. official data portals, benchmark pages, repository documentation, or methodological standards
4. influential working papers only when the field commonly uses them and they are clearly relevant
5. expert commentary only as a pointer to primary sources

Do not rely on generic blogs or secondary summaries when a primary source is available.

## Canonical and Frontier Balance

The field memo should include both of these:

- canonical works that define the field's basic vocabulary, assumptions, and classic questions
- frontier papers from recent years that show where the field is actively moving

A useful default is:

- 3 to 5 canonical sources
- 5 to 8 frontier sources

Adjust upward only when the topic is broad.

## Discovery Workflow

1. Identify the discipline and probable subfield.
2. Search for survey papers or handbook chapters to build vocabulary.
3. Identify recurring top venues or institutions.
4. Search within those venues for recent influential papers.
5. Track recurring datasets, methods, and debates.
6. Stop when the field map stabilizes and new sources mostly repeat known patterns.

## Retention Rules

Keep a source when at least one of these is true:

- it defines the core problem
- it is repeatedly cited as foundational
- it introduces a method still used by later work
- it represents a major benchmark, dataset, or protocol
- it captures a current frontier argument or limitation
- it directly informs the user's possible research question

Drop a source when it is:

- only tangentially related
- low credibility or unclear provenance
- redundant with stronger sources
- too old to represent the frontier when the frontier matters
- too new and too thin to trust without corroboration

## Writing Norm Discovery

Infer writing norms from the field rather than assuming them. While reading, note:

- common section order
- how papers state research questions
- how claims are hedged
- whether theory, experiments, or empirical strategy carries the paper
- what counts as a convincing contribution
